Skin Colors

Create & apply your own skin color
1. Find a color picker on web such as: http://html-color-codes.info/
2. Choose your color and copy the generated code, in example #ffd700
3. Open assets/css/skins/mycolor.css and replace the color code: #ff692a with the color generated at step 2. Save the file.
4. Go to index.html and right after 
<link href="style.css" rel="stylesheet">
add : 
<link href="assets/css/skins/mycolor.css" rel="stylesheet">

Save. Open your index.html and see the changes.
Repeat step 4 with every page where you want to operate the change.

TIP: You can create as many skin colors as you wish. Just copy "mycolor.css" file, rename it (i.e.green.css). Replace the color code with the desired one then open any .html file and add the code from step 3, just make sure that the name matches. In our example, for green.css , our path would be:
